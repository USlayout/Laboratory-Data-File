# Laboratory-Data-File

ベンチマークや測定データを共有するリポジトリです。データの再利用・比較がしやすいよう、以下のルールに従ってください。

- **命名ルール**: ファイル／フォルダ名に「内容-YYYYMMDD[-HHMM]」を含める。例: `sysbench-cpu-performance-20260101-0000`。
- **必須メタ情報**: 各データセットには測定日・測定内容・実行コマンド／スクリプト・実行環境（OS・バージョン・ハードウェア）を記載する（README、metadata.yaml、または同梱のTXT/MD）。
- **注意事項**: 個人情報や機密情報は含めないこと。公開前に必ず内容を確認する。
- `benchmark.sh` の設定値は同階層の `benchmark.conf` に分離しています。必要に応じてそこを編集してください。
